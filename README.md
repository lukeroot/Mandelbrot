# Mandelbrot
Interactive Mandelbrot. Currently messy code but feel free to run

`sudo apt-get install python-pygame`

### Controls
* __1__ 		Improve display (Will lag)
* __2__ 		Reverse 1
* __q__ 		Quit
* __arrows__	Move around
* __p__		Zoom in
* __l__		Zoom out

You'll find some strange behaviour with the keys
